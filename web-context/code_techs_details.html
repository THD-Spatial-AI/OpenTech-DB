Technical Description & Methodology

Data represents utility-scale photovoltaic systems using crystalline silicon technology. CAPEX includes module, inverter, structural, and electrical balance of system (BOS) components. Metrics are normalized for standard conditions in temperate climate zones. WACC represents post-tax nominal rates used for LCOE calculation. Degradation assumes linear reduction in power output over the technical lifetime.
